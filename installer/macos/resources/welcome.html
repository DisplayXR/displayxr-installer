DisplayXR

This installer will set up DisplayXR — the OpenXR runtime for 3D displays — on your Mac.

What gets installed:

  • DisplayXR Runtime — OpenXR runtime with Vulkan compositor and MoltenVK, registered as the system’s active OpenXR runtime.
  • Sim Display plug-in — vendor-neutral display processor for use without dedicated 3D display hardware.

Additional components (DisplayXR Shell, Leia SR plug-in, MCP Tools) will be added to the macOS bundle as they publish macOS packages. Today these ship for Windows only.

To uninstall, run /Library/Application Support/DisplayXR/uninstall.sh.
